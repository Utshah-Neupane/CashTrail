1. Backend Folder

    Creates package.json:
    npm init -y

    To install express:
    npm i express

    To change version of express from 5 to 4 because it is more stable:
    npm i express@4.21.0

    This then:
    npm i dotenv@16.5.0 cors@2.8.5  @neondatabase/serverless@1.0.0

    Then for bakend folder, to run, I did:
    node server.js

    So that we don't have to restart and kill terminal everytime
    npm i -D nodemon

    Installing the ratelimiting package:
    npm i @upstash/redis@1.34.9 @upstash/ratelimit@2.0.5

    For running the program:
    npm run dev






2. Mobile folder

    Creating the template app:
    npx create-expo-app@latest .

    To reset the project
    npm run reset-project

    To generate react-native functional component
    rnfe

    Then to run the app:
    npx expo

    For authentication using clerk:
    npm install @clerk/clerk-expo
    npm install expo-secure-store

    Keyboard avoiding/blocking view package:
    npm i react-native-keyboard-aware-scroll-view
